# SmartPredict AI 🚀

An end-to-end Machine Learning system that automates:

- Data Preprocessing
- Feature Engineering
- Model Training (All Major Models)
- Model Evaluation
- Best Model Selection
- Deployment via Streamlit

## Features
- Supports Classification & Regression
- Auto ML Pipeline
- Interactive UI
- Model Saving

## Run Project

```bash
pip install -r requirements.txt
streamlit run app.py
